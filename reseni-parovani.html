Řešený příklad Ř05

Párování a generická hodnost

Tři úlohy nad bipartitními grafy: generická hodnost matice přes Hallovu větu a dvakrát optimální (vážené) párování Kuhnovým–Munkresovým (maďarským) algoritmem.

1 Generická hodnost matice

Lze zvolit čísla a, …, s tak, aby matice A byla regulární?

y1 y2 y3 y4 y5 y6 x1 [ a 0 b 0 0 c ] x2 [ 0 d 0 e f g ] x3 [ h 0 i 0 0 0 ] x4 [ 0 0 j 0 0 k ] x5 [ l m n o p 0 ] x6 [ r 0 0 0 0 s ]
Kritérium

Matice A je genericky regulární právě tehdy, když v jejím bipartitním grafu B(A) (řádky ↔ sloupce, hrana = nenulový prvek) existuje perfektní párování. Maximální generická hodnost gh(A) rovná se velikosti maximálního párování v B(A).

Sestrojíme B(A). Zvýrazněné hrany tvoří maximální párování (5 hran), zakroužkovaná množina S porušuje Hallovu podmínku:

Hallova podmínka. Vezmeme S = {x1, x3, x4, x6}. Jejich sousedé:

N(x1)={1,3,6}, N(x3)={1,3}, N(x4)={3,6}, N(x6)={1,6} N(S) = {1, 3, 6} → |N(S)| = 3 < 4 = |S|

Hallova podmínka je porušena, takže perfektní párování neexistuje a A nemůže být genericky regulární.

Výsledek

Maximální párování má 5 hran (např. x3–1, x4–3, x6–6, x2–4, x5–5), proto gh(A) = 5. Matice regulární být nemůže.

Postup

1. Počáteční přípustné ohodnocení: ℓ(xᵢ) = max řádku (sloupec vpravo), ℓ(yⱼ) = 0.
2. Graf rovnosti G_ℓ = hrany, kde ℓ(xᵢ)+ℓ(yⱼ) = wᵢⱼ.
3. Maďarský algoritmus hledá rozšiřující cesty; když narazí na množinu S s |N(S)| < |S|, opraví ohodnocení o α = min{ℓ(xᵢ)+ℓ(yⱼ)−wᵢⱼ} (přes xᵢ∈S, yⱼ∉T) — zmenší na S, zvětší na T.
4. Opakuje, až je párování v G_ℓ perfektní — to je optimum.

Výsledek

Cena M = 5 + 7 + 3 + 4 + 6 + 5 = 30.

Na co dát pozor

  • Generická hodnost: hledá se největší párování v B(A); regularita ⇔ perfektní párování. Neregularitu dokazuje jedna Hallova porušující množina S.
  • Při hledání S: začni od nepokrytého uzlu, střídej nepárovací/párovací hrany; S jsou dosažené uzly v X, T v Y.
  • Kuhn–Munkres řeší maximalizaci ceny. Pro minimalizaci převeď wᵢⱼ → C − wᵢⱼ.
  • Invariant: ℓ(xᵢ)+ℓ(yⱼ) ≥ wᵢⱼ stále platí; oprava o α přidá do G_ℓ aspoň jednu novou hranu.
  • Cena optimálního párování = Σ ℓ(xᵢ) + Σ ℓ(yⱼ) (duální kontrola výsledku).
