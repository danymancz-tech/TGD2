Řešený příklad Ř04

Dualita úloh LP

Ke každé úloze LP existuje úloha duální. Ukážeme, jak ji sestavit, a na čtyřech příkladech ověříme větu o dualitě — včetně mezních případů, kdy úloha nemá řešení.

Připomenutí — jak vzniká duál

Primární úloha min cᵀx, Ax = b (resp. ≥ b), x ≥ 0. Duální úloha má pro každou rovnici primáru jednu proměnnou a pro každou proměnnou primáru jednu nerovnici:

primár min cᵀx, Ax = b, x ≥ 0 duál max bᵀy, Aᵀy ≤ c, y ∈ ℝ (volné, protože rovnice)

Rovnice v primáru ⇒ volná duální proměnná; nerovnice ≥ ⇒ y ≥ 0.

Věta o dualitě

Slabá: pro každé přípustné x (primár) a y (duál) platí cᵀx ≥ bᵀy.
Silná: má-li jedna úloha optimum, má je i druhá a ceny se rovnají.
Důsledek pro mezní případy: je-li primár neomezený, je duál nepřípustný (prázdná množina) — a naopak.

1 Dualita 1 — rovnostní omezení

Příklad 1.1 — obě úlohy mají optimum

Primár: min x + y 2x + y − z = 2 x, y, z ≥ 0

Všechny relativní ceny ≤ 0 → optimum. Bázická x = 1, nebázické y = z = 0.

Primár — optimum

x = 1, y = 0, z = 0, cena x + y = 1.

Sestavení duálu. Jedna rovnice ⇒ jedna volná proměnná y. Každá primární proměnná dá jednu nerovnici (sloupec Aᵀ ≤ příslušná cena):

x: 2y ≤ 1 y: y ≤ 1 z: −y ≤ 0 ──────────────── Duál: max 2y, 0 ≤ y ≤ ½

Maximum 2y na [0, ½] nastane v y = ½, cena 2·½ = 1.

Ověření duality

Cena primáru = cena duálu = 1. ✓ Silná dualita platí.

Příklad 1.2 — neomezený primár, prázdný duál

Stejná množina přípustných řešení, jen jiná cena:

Primár: min x − y 2x + y − z = 2 x, y, z ≥ 0

Do báze má jít z (kladná relativní cena +1), ale ve sloupci z je jediný prvek −1 < 0 — poměrový test selže, žádná mez. Cenu lze zmenšovat donekonečna:

Primár — neomezený

Cenová funkce není zdola omezená na množině přípustných řešení.

Duál podle stejného receptu:

x: 2y ≤ 1 y: y ≤ −1 z: −y ≤ 0 (tj. y ≥ 0) ──────────────── Duál: max 2y, 0 ≤ y ≤ −1 ← spor
Pointa

Podmínka 0 ≤ y ≤ −1 je nesplnitelná → duál je prázdný. To přesně odpovídá větě: neomezený primár ⇔ nepřípustný duál.

2 Dualita 2 — nerovnostní omezení

Příklad 2.1 — obě úlohy mají optimum

Primár: min x + y + 2z x + y + z ≥ 1 x + 4z ≥ 2 x, y, z ≥ 0

Převedeme na standardní tvar (odečteme přebytkové proměnné s₁, s₂), najdeme výchozí PBR a simplexem dojdeme k optimu. Klíčové mezikroky:

  • Výchozí PBR (0, ½, ½, 0, 0) má cenu 3/2 — není optimální.
  • Do báze vstoupí x, odejde y.
Primár — optimum

x = 2/3, y = 0, z = 1/3, cena 2/3 + 0 + 2·(1/3) = 4/3.

Duál. Dvě nerovnice ≥ ⇒ dvě nezáporné proměnné u, v:

x: u + v ≤ 1 y: u ≤ 1 z: u + 4v ≤ 2 ──────────────── Duál: max u + 2v, u, v ≥ 0

Tato úloha má hned vidět výchozí PBR (pomocné proměnné), optimalizací dostaneme:

Duál — optimum

u = 2/3, v = 1/3, cena u + 2v = 2/3 + 2/3 = 4/3.
Kontrola: u+v = 1 (těsně), u+4v = 2 (těsně).

Ověření duality

Cena primáru = cena duálu = 4/3. ✓

Příklad 2.2 — neomezený primár, prázdný duál

Primár: min x − y x + y + z ≥ 1 x + 4z ≥ 2 x, y, z ≥ 0

Proměnnou y lze zvětšovat libovolně (omezení ≥ přitom zůstávají splněna), takže x − y → −∞:

Primár — neomezený

Množina přípustných řešení není omezená v kladném směru osy y, cena není zdola omezená.

Duál: max u + 2v u + v ≤ 1 u ≤ −1 u + 4v ≤ 0 u, v ≥ 0 → 0 ≤ u ≤ −1 ← spor
Pointa

Opět prázdný duál (z u ≤ −1 a u ≥ 0), což odpovídá neomezenému primáru.

Na co dát pozor

  • Recept na duál: rovnice → volná proměnná, nerovnice ≥ → proměnná ≥ 0; počet duálních proměnných = počet primárních omezení.
  • Sloupce matice A se v duálu stávají řádky (Aᵀ), pravé strany a ceny si vymění role.
  • Silná dualita: mají-li obě úlohy optimum, jejich ceny se rovnají — užitečná kontrola výpočtu.
  • Mezní případy: neomezený primár ⇔ nepřípustný duál. Pozor: prázdný primár nemusí znamenat neomezený duál (může být taky prázdný).
  • Komplementarita: v optimu jsou těsné právě ty duální nerovnice, jejichž primární proměnná je kladná (viz u+v=1, u+4v=2 u 2.1).
