Otázka 07 · Lineární programování

Dualita úloh LP

Ke každé (primární) úloze LP existuje duální úloha. Slabá a silná věta o dualitě dávají dobrou charakteristiku optimality.

7.1 Primární a duální úloha

Vyjdeme z kanonické primární úlohy a vynásobíme Ax ≥ b nezáporným vektorem y ≥ 0 zleva. Pokud ATy ≤ c, dostaneme dolní odhad bTy ≤ cTx. Nejlepší dolní odhad hledáme maximalizací.

Dvojice duálních úloh
Primární (1)Duální (2)
min cᵀxmax bᵀy
Ax ≥ bAᵀy ≤ c
x ≥ 0y ≥ 0

Duální úloha k duální je opět primární — proto vzájemně duální úlohy.

7.2 Věty o dualitě

Věta 3.1 — slabá věta o dualitě

Pro každé přípustné x primární a každé y duální úlohy platí bTy ≤ cTx.

Věta 3.2 — silná věta o dualitě

Vektory x, y jsou optimální právě když bTy = cTx.

Důsledky
  • 3.1: rovnost bᵀy = cᵀx je postačující pro optimalitu.
  • 3.2: je i nutná.
  • 3.2: je-li cena primární zdola neomezená, je duální prázdná (a naopak).
  • 3.4: mají-li obě úlohy přípustné řešení, mají obě optimum a rovnají se.

7.3 Duální úlohy v jiných tvarech

PrimárníDuální
min cᵀx, Ax = b, x ≥ 0max bᵀy, Aᵀy ≤ c, y ∈ ℝᵐ
max cᵀx, Ax = b, x ≥ 0min bᵀy, Aᵀy ≥ c, y ∈ ℝᵐ
Rozdíl u standardního tvaru

U rovnosti Ax = b nepotřebujeme y ≥ 0 — duální proměnná může být libovolná z ℝm.

7.4 Ekonomická interpretace — stínové ceny

U úlohy o dietě: výrobce chce prodávat syntetické vitamíny. yi = cena jednotky i-tého vitamínu, bTy = celkový příjem, podmínka ATy ≤ c garantuje konkurenceschopnost. Duálním proměnným se říká stínové ceny.

7.5 Dualita jako dobrá charakteristika

Proč je to důležité

Rozhodovací úloha „je dané PBR x₀ optimální?“ je v NP ∩ co-NP:

  • Není-li optimální → najdeme lepší PBR (pozitivní argument).
  • Je-li optimální → najdeme přípustné y₀ duální úlohy s cᵀx₀ = bᵀy₀.

Pro optimální tok je analogickou dobrou charakteristikou věta o potenciálech (viz otázka 15).

Co umět ke zkoušce

  • Konstrukci duální úlohy z primární (vynásobení y ≥ 0, odvození odhadu).
  • Tabulku primární ↔ duální v kanonickém i standardním tvaru.
  • Slabou (3.1) a silnou (3.2) větu o dualitě + důsledky 3.1–3.4.
  • Stínové ceny (ekonomická interpretace).
  • Dualita ⟹ LP je dobře charakterizovaná (NP ∩ co-NP) — propojení s otázkou 15.
