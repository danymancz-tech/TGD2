Řešený příklad Ř01

Optimální tok metodou potenciálů

Najdeme nejlevnější přípustný tok v síti se třemi uzly. Stejnou síť pak vyřešíme podruhé převodem na úlohu lineárního programování — výsledek musí vyjít stejně.

Značení hran

Každá hrana je popsána ve tvaru tok(cena, propustnost), tedy x(c, r). Cena je za jednotku toku, propustnost je horní mez. Cílem je minimalizovat Σ c·x při dodržení bilancí v uzlech a mezí 0 ≤ x ≤ r.

A Zadání

Síť má tři uzly s bilancemi +10, −3, −7 (kladná = zdroj, záporná = spotřebič; součet je 0, síť je vyvážená) a tři hrany:

B Metoda potenciálů

hrana opory nasycená hrana ostatní hrana
  1. Výchozí přípustný tok Zvolíme libovolný přípustný tok, např. x₁=4, x₂=6, x₃=1. Ověříme bilance: 4+6=10 ✓, 4−1=3 ✓, 6+1=7 ✓. Cena je 3·4 + 7·6 + 3·1 = 57. Hrana A→C je nasycená (x₂=6=r), zbylé dvě hrany mají tok ostře uvnitř mezí — tvoří oporu (kostru sítě).
  2. Potenciály na opoře a kontrola optimality Položíme u_A = 0 a podél hran opory dopočítáme potenciály tak, aby potenciál narostl po hraně přesně o její cenu:
    u_B = u_A + c(A→B) = 0 + 3 = 3 u_C = u_B + c(B→C) = 3 + 3 = 6
    Na neoporové (nasycené) hraně A→C teď spočítáme redukovanou cenu c̄ = c(A→C) − (u_C − u_A) = 7 − (6 − 0) = +1. U nasycené hrany žádá optimalita c̄ ≤ 0; tady je +1 > 0 ⇒ spor. Hrana A→C je „příliš drahá“ — chceme po ní posílat méně.
  1. Zlepšující cyklus (ZRP) Tok z drahé přímé hrany A→C (cena 7) odkloníme na levnější cestu A→B→C (cena 3+3=6) — úspora 1 na jednotku. O kolik můžeme tok přesunout? Limituje nás rezerva podél cyklu:
    Δ = min( r(A→B)−x₁, r(B→C)−x₃, x₂ ) = min(6−4, 4−1, 6) = 2
    Posuneme Δ=2: x₁ a x₃ vzrostou o 2, x₂ klesne o 2.
  2. Nový tok a kontrola Dostáváme x₁=6, x₂=4, x₃=3 s cenou 3·6 + 7·4 + 3·3 = 55 (ušetřili jsme 2 = Δ·1). Přepočítání potenciálů už nedá spor ⇒ tok je optimální.

C Kontrola převodem na LP

Tutéž síť lze zapsat jako úlohu LP: proměnné jsou toky po hranách, rovnice jsou bilance uzlů, nerovnosti jsou propustnosti. Třetí bilanci vynecháme (je lineární kombinací zbylých dvou).

min 3x₁ + 7x₂ + 3x₃ x₁ + x₂ = 10 x₁ − x₃ = 3 x₁ ≤ 6, x₂ ≤ 6, x₃ ≤ 4 x₁, x₂, x₃ ≥ 0

Z rovnic plyne x₂ = 10 − x₁ a x₃ = x₁ − 3, takže cílová funkce závisí jen na x₁:

3x₁ + 7(10−x₁) + 3(x₁−3) = −x₁ + 61 ⇒ minimum při co největším x₁.

Meze dávají x₁ ≤ 6 a x₃ = x₁−3 ≤ 4 ⇒ x₁ ≤ 7, navíc x₂ = 10−x₁ ≤ 6 ⇒ x₁ ≥ 4. Největší přípustné je x₁ = 6, tedy x₂ = 4, x₃ = 3.

Výsledek Optimální tok x₁ = 6, x₂ = 4, x₃ = 3 s cenou 55. Obě metody — potenciály i LP — dávají totéž.

Na co dát pozor

  • Potenciál roste po hraně o cenu; orientace hran je klíčová.
  • Optimalita: u nenasycené hrany žádáme c̄ ≥ 0, u nasycené c̄ ≤ 0, na opoře c̄ = 0.
  • Rezerva cyklu = nejmenší ze zbývajících kapacit „nahoru“ a toků „dolů“ podél cyklu.
  • Při převodu na LP vždy vypustíme jednu bilanci (jsou lineárně závislé).
