Otázka 01 · Optimální tok

Optimální tok v síti

Síť s více zdroji a stoky, cenou na hranách. Hledáme tok minimální ceny. Dvě charakterizační věty: o záporných rezervních polocyklech a o potenciálech.

1.1 Formulace úlohy

Máme síť G⃗ s propustností hran rij a intenzitami uzlů ai. Navíc každé hraně přiřadíme cenu cij ∈ ℝ (může být i záporná).

Definice — cena toku

Je-li x tok v G⃗, pak číslo

c(x) = Σ(i,j)∈H(G⃗) cij · xij

je cena toku x. Tok s minimální cenou je optimální tok.

Aplikace

Plánování přepravy — c(x) = celkové přepravní náklady, optimální tok = nejlevnější rozvozní plán. Úloha o skladu — zde ceny mohou být i záporné (prodej vs. nákup), což ospravedlňuje předpoklad cij ∈ ℝ.

1.2 Záporné rezervní polocykly

V symetrizaci grafu pracujeme s polocestami a polocykly. Hrany ve směru obíhání jsou souhlasné, opačné nesouhlasné.

Definice — cena polocyklu
c(P) = Σsouhlasné c(h) − Σnesouhlasné c(h)
Definice — zvětšení toku

Tok x lze na polocyklu C zvětšit o δ, jestliže

  • xij + δ ≤ rij pro každou souhlasnou hranu,
  • xij − δ ≥ 0 pro každou nesouhlasnou hranu.

1.3 Charakterizační věta o ZRP

Věta 1.1

Tok x je optimální, právě když jej nelze zvětšit na žádném polocyklu záporné ceny.

Pojem ZRP

Polocyklus záporné ceny, na němž lze tok zvětšit, se nazývá záporný rezervní polocyklus (ZRP). Věta tedy říká: tok je optimální ⟺ neexistuje pro něj ZRP.

Idea důkazu (←): Není-li x optimální, existuje lepší tok y. Cirkulaci z = y − x rozložíme na toky v cyklech; protože c(y) < c(x), některý odpovídající polocyklus musí být záporný.

1.4 Věta o potenciálech

Věta 1.3 — potenciály

Tok xij je optimální, právě když existuje ohodnocení uzlů čísly Vi tak, že:

Vj − Vi ≤ cij jestliže xij = 0 Vj − Vi = cij jestliže 0 < xij < rij Vj − Vi ≥ cij jestliže xij = rij

Opora toku G⃗x = faktor tvořený hranami s 0 < xij < rij. Potenciály se konstruují postupně: zvolíme Vi₀ = 0 a šíříme podél hran opory (Vj = Vi + cij).

Co umět ke zkoušce

  • Formulace ceny toku a definice optimálního toku.
  • Věta 1.1 (ZRP) — znění + idea důkazu přes cirkulaci z = y − x.
  • Věta 1.3 (potenciály) — tři podmínky podle vztahu xij k rij.
  • Rozdíl: ZRP není dobrá charakteristika sám o sobě; věta o potenciálech je (viz otázka 15).
  • Souvislost: nalezení ZRP → hledání záporného cyklu → Floydův algoritmus (viz otázka 02).
